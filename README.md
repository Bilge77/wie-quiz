# WIE Quiz – International Day of Women and Girls in Science

This project is a web-based quiz application developed for the IEEE Women in Engineering (WIE) event, celebrating the International Day of Women and Girls in Science.

## Project Purpose
The aim of this project is to create an interactive and engaging quiz experience to raise awareness about women's contributions to science and engineering.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Multiple-choice quiz questions
- Interactive user interface
- Instant feedback for answers
- Simple and responsive design

## Project Structure
- `index.html` → Main structure of the application  
- `style.css` → Styling and layout  
- `script.js` → Quiz logic and interactions  
- `assets/` → Images and additional resources
